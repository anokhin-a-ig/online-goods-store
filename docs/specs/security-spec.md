# Безопасность

## Аутентификация

- **Spring Security** через Form Login
- Пароли: **BCrypt** (Spring Security `PasswordEncoder`)
- Сессии (не JWT)

## Требования к паролю

- Минимум 8 символов
- Хотя бы одна цифра
- Хотя бы одна латинская буква

## Блокировка при неудачных входах

- Поля в User: `failedAttempts` (int), `lockTime` (LocalDateTime)
- После **5 неудачных** попыток → `lockTime = now + 15 min`
- При успешном входе: сброс `failedAttempts = 0`, `lockTime = null`
- Попытка входа при заблокированном пользователе: отказ

## Роли и права

- `ROLE_USER` — базовый доступ (каталог, корзина, заказы)
- `ROLE_ADMIN` — админ-панель (`/admin/**`)
- Доступ через `@PreAuthorize` и `hasRole()`

## CSRF

- Включён по умолчанию
- REST эндпоинты (если есть) — CSRF отключён или через токен

## CORS

- Разрешён `localhost:8080` по умолчанию
- Настройка через `WebMvcConfigurer`

## Rate Limiting

- На эндпоинт `/login`: не более 10 попыток в минуту с одного IP
- Реализация: Bucket4j или через `OncePerRequestFilter` с подсчётом

## Защита

- XSS: Thymeleaf экранирует по умолчанию
- SQL-инъекции: JPA/Spring Data параметризованные запросы
- Доступ к статике: без аутентификации
- Актуатор: только для ADMIN
