# Административная панель

## Доступ

Только для `ROLE_ADMIN`. Все эндпоинты под префиксом `/admin`.

## Функциональность

### Управление категориями (CRUD)
- Создание/редактирование категории (name, description, parent)
- Деактивация категории (soft-delete через active)
- Каскадная деактивация дочерних при деактивации родительской
- Список всех категорий (включая неактивные)

### Управление товарами (CRUD)
- Создание/редактирование товара (name, description, price, stock, category, image)
- Загрузка изображения (JPEG/PNG, макс 5MB, в uploads/)
- Soft-delete (active = false)
- Список всех товаров с фильтрами

### Управление заказами
- Просмотр всех заказов с фильтром по статусу и пользователю
- Изменение статуса заказа:
  - CREATED → PAID, SHIPPED, DELIVERED, CANCELLED
  - PAID → SHIPPED, DELIVERED
  - SHIPPED → DELIVERED
- Отмена заказа (CANCELLED) с возвратом остатков

### Управление пользователями
- Список всех пользователей
- Блокировка/разблокировка (blocked)
- Смена роли (USER ↔ ADMIN)

## Безопасность

- `@PreAuthorize("hasRole('ADMIN')")` на всех методах админ-контроллера
- Thymeleaf шаблоны: extends layout с проверкой роли
- CSRF защита активна
