# Стек используемых в проекте технологий

## Технологии

### Программирование
- Java 17
- Maven
- Spring Boot 3.5.X
- Spring Validator
- Spring Data Jpa
- Spring Security
- record классы для DAO/DTO
- Lombok

### Страницы, представления
- Thymeleaf
- html
- css

### Тестирование
- testcontainers
- Mockito
- JUnit5

### Контейнеризация
- Docker
- docker-compose
- dockerfile

### База данных
- postgres 15:alpine
- liquibase

### Документировани
- Javadoc для всех методов кроме контроллеров
- Swagger для контроллеров

### Передача данных между страницами и бэкендом
- используется record классы
